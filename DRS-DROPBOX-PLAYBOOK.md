# DRS Dropbox Organization Playbook

## Core Folder Types

- **Projects** — real DRS project folders, active/in-progress/completed.
- **Prelims / Proposals** — inquiries, pursuits, and opportunities before they become full projects.
- **Operations** — HR/legal/office/admin material. Treat as restricted by default.
- **Engineering** — engineering templates and reference material.
- **Design** — CAD/design-related material.

## Project Folder Naming

Typical project folders use:

```text
YYYY-### Project Name - Client Name
```

Example:

```text
2026-118 1028 Ridgedale Dr - Titan
```

## Project Status

Active and completed projects may be mixed together in the project folder area.

Do **not** assume a project is active or inactive based only on where it appears.

## Common Project Subfolders

A project may contain some or all of these:

- Contractor Submittals
- Correspondence
- Design
- Drawings
- Incoming Docs / Incoming
- Outgoing
- Photos
- Preliminary Design
- Proposal & Estimate
- RFI
- Soil Report
- Submittals
- Testing

## Default Filing Behavior

When adding documents to a project:

1. Require the project to be identified first.
2. If the requester specifies a destination subfolder, use that existing location.
3. If no destination is specified, place files in the project’s existing `Incoming` / incoming-docs folder.
4. Use a dated subfolder format:

```text
YYYYMMDD Client Name
```

Example:

```text
20260514 Titan
```

If the client name cannot be determined reliably, ask before filing.

## Safety Rules for AI Agents

- Treat Dropbox as a source system.
- Read/browse first.
- Do not delete Dropbox files or folders.
- Do not move, rename, overwrite, or reorganize files unless explicitly approved.
- Do not create new project folders unless explicitly asked.
- Treat Operations folders as sensitive/restricted.
- Do not summarize or expose HR, legal, finance, payroll, owner compensation, credentials, or private admin material unless specifically authorized.

## Good Agent Behavior

- Search by project number first when available.
- If no project number is available, search by address, client, contractor, or project name.
- Prefer existing folder structure over inventing new folders.
- When uncertain, ask a clarifying question instead of guessing.
